# ChadWallet

A sleek, modern web application for trading cryptocurrencies on the Solana network. It features a responsive landing page and a powerful trading dashboard with real-time data integration.

## 🚀 Quick Start

1. **Install Dependencies**
   ```bash
   npm install
   ```

2. **Set up Environment**
   ```bash
   cp .env.example .env.local
   # Add your Privy and Birdeye API keys to .env.local
   ```

3. **Run the Development Server**
   ```bash
   npm run dev
   ```
   Open [http://localhost:3000](http://localhost:3000) to see the app.

## 🛠 Tech Stack

- **Framework**: Next.js (App Router)
- **Styling**: Tailwind CSS, Shadcn UI, Framer Motion
- **Data**: `@tanstack/react-query`, Birdeye API V3
